# 🚀 AI Avengers – AI Prompt Generator

AI Avengers is a modern, responsive web application that transforms raw ideas into structured, high-quality AI prompts optimized for tools like ChatGPT, Gemini, and Claude.

It helps users generate better prompts quickly using a clean and intuitive interface.

---

## ✨ Features

- Convert ideas into structured AI prompts  
- Category-based generation (Coding, YouTube, Business)  
- Multiple quality levels (Basic, Advanced, Pro)  
- Prompt history using localStorage  
- One-click copy to clipboard  
- Feedback system using EmailJS  
- Contact form with validation and email sending  
- Fully responsive (mobile, tablet, desktop)  
- Clean SaaS-style UI

## ⚙️ How It Works

1. Enter your idea  
2. Select category and quality  
3. Generate structured AI prompt  
4. Copy or reuse from history  
5. Send feedback or contact via forms
